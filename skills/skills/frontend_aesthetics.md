<frontend_aesthetics>
Your goal is to create distinctive, high-quality, non-generic frontend design.
Avoid converging to common AI patterns such as:
- Inter, Roboto, Open Sans, system fonts
- Purple gradients on flat white backgrounds
- Cookie-cutter hero sections
- Boring evenly spaced components

Focus on the following 4 design pillars:

1. TYPOGRAPHY
- Use bold, character-rich fonts with strong identity.
- Recommended families:
  * Headings: Space Grotesk, Clash Display, or Playfair Display
  * Body: IBM Plex Sans, Inter Tight, or Source Sans 3
  * Code/mono accents: JetBrains Mono or Fira Code
- Make decisive choices: strong weight contrast, large type hierarchy, dramatic spacing.

2. COLOR & THEME
- Commit to a cohesive dark theme using:
  * True Black (#000000)
  * Electric Purple (#6A0DFF or #7C3AED)
  * White (#FFFFFF)
  * Optional neon accents (#A855F7)
- Use layered gradients, subtle glows, and high contrast.
- Avoid timid, washed-out palettes.
- Utilize CSS variables for consistent theming.

3. MOTION & INTERACTION
- Introduce meaningful motion:
  * Staggered fade-ins on page load
  * Hover micro-interactions
  * Subtle glowing transitions
- Prefer CSS keyframes for HTML; use Framer Motion for React.
- One polished animation is better than many noisy ones.

4. BACKGROUND & DEPTH
- Never use flat colors alone.
- Add depth with:
  * Radial gradients
  * Soft grain/noise
  * Faint grid/mesh patterns
  * Layered geometric shapes
- Maintain UW Blockchain’s “builder, futuristic, hacker” tone.

CREATIVE GUIDANCE
- Always include intentional personality in the design.
- Interpret the theme based on context instead of repeating patterns.
- Avoid falling into new repetitive defaults (like always choosing Space Grotesk).
- Think like a designer, not a token sampler.

Your purpose: generate frontends that look handcrafted, modern, and aligned with black–purple–white UW Blockchain branding.
</frontend_aesthetics>
